❌ Your browser does not support WebGPU

Try the latest version of Chrome or see a list of supported browsers here.

❌ Unable to request WebGPU adapter

Your browser supports WebGPU, but was unable to access your device's GPU. If using Chrome, try the following:

  • Make sure Chrome's "Use graphics acceleration when available" setting is turned on.
  • Try turning on the chrome://flags/#enable-unsafe-webgpu flag. If on Linux, also turn on the chrome://flags/#enable-vulkan flag. Then restart Chrome.

Another possibility is that your GPU hardware is incompatible with WebGPU.
